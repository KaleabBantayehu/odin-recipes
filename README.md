# Odin Recipes

A simple recipe website built as part of The Odin Project Foundations course.

## About

This project was created to practice the basics of web development using HTML and CSS.

The website includes:

- Home page
- Lasagna recipe
- Pizza recipe
- Burger recipe

## Technologies Used

- HTML5
- CSS3

## What I Learned

While building this project I practiced:

- HTML page structure
- Links and navigation
- Images
- Lists
- CSS styling
- Basic page layout

## Future Improvements

- Improve the page layout using Flexbox or Grid
- Make the website responsive
- Improve the overall design
- Add more recipes
